# security-alert-issues-bot

> A GitHub App built with [Probot](https://github.com/probot/probot) that [Probot] Sync GitHub Security Alerts and GitHub Issues.

## Setup

```sh
# Install dependencies
npm install

# Run typescript
npm run build

# Run the bot
npm start
```
